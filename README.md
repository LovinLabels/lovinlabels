# LovinLabels Website

This is the official website for LovinLabels, a brand focused on Montessori toys for children with autism.

## How to Deploy on GitHub Pages
1. Upload the files to a GitHub repository.
2. Go to Repository Settings > Pages.
3. Set the branch to 'main' and save.
4. Your website will be live at `https://yourusername.github.io/lovinlabels/`.

## How to Deploy on Netlify
1. Create an account at [Netlify](https://www.netlify.com/).
2. Upload the files or connect a GitHub repository.
3. Set the domain in Netlify's settings.
4. The site will be live with your custom domain `lovinlabels.com`.
